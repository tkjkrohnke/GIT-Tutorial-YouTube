# Grafischer Taschenrechner in Python

Dies ist ein einfacher grafischer Taschenrechner, erstellt mit Tkinter in Python. Er unterstützt grundlegende arithmetische Operationen: Addition, Subtraktion, Multiplikation und Division.

## Funktionen

- Zahlen: 0-9
- Dezimalpunkt: .
- Operatoren: +, -, *, /
- Gleich: =
- Clear: C (löscht die Eingabe)

Der Taschenrechner kann nun komplexe Ausdrücke wie 2+3*6 berechnen (ergibt 20).

## Wie man es startet

1. Stelle sicher, dass Python installiert ist (Version 3.x).
2. Öffne ein Terminal im Projektverzeichnis.
3. Führe den Befehl aus: `python calculator.py`

Ein Fenster mit dem Taschenrechner wird geöffnet.

## Verwendung

- Klicke auf Zahlen und Operatoren, um eine Berechnung einzugeben.
- Drücke =, um das Ergebnis zu berechnen.
- Drücke C, um zu löschen.

## Beispiel

- Eingabe: 5 + 3 =
- Ergebnis: 8.0

Hinweis: Division durch Null zeigt "Error" an.
